=========================================================
INSTRUKSI UNTUK USER
=========================================================

STEP 1: Download & Install
---------------------------
1. Download Windows Executor V2-Setup-1.0.0.exe
2. Double-click installer
3. Pilih lokasi install (default: C:\Program Files\Windows Executor V2)
4. Klik Install
5. Tunggu sampai selesai (~2-3 menit)

STEP 2: Konfigurasi Awal
---------------------------
1. Buka aplikasi (dari Desktop shortcut atau Start Menu)
2. Aplikasi akan create file .env otomatis
3. WAJIB isi 3 field ini di file .env:
   
   WE_V2_API_KEY=your_api_key_here
   WE_V2_API_SECRET=your_api_secret_here
   WE_V2_EXECUTOR_ID=executor_001

4. Restart aplikasi

STEP 3: Connect MT5
---------------------------
1. Pastikan MetaTrader 5 sudah terinstall
2. Login ke MT5 account
3. Aplikasi akan auto-detect MT5
4. Backend start otomatis (tunggu 10-15 detik)

STEP 4: Mulai Trading
---------------------------
1. Buka tab "Strategies"
2. Lihat strategi dari platform di "Available Strategies"
3. Klik "Start Strategy" untuk mulai
4. Monitor di tab "Positions" dan "Dashboard"

=========================================================
TROUBLESHOOTING UMUM
=========================================================

ISSUE: Backend tidak start
SOLUSI:
- Check port 8081 tidak dipakai aplikasi lain
- Check .env file ada dan configured
- Check logs di: %LOCALAPPDATA%\WindowsExecutorV2\logs\

ISSUE: MT5 tidak terdetect
SOLUSI:
- Pastikan MT5 terinstall
- Coba set WE_V2_MT5_PATH manual di .env
- Restart aplikasi

ISSUE: Strategy tidak bisa start
SOLUSI:
- Pastikan MT5 sudah login
- Check API credentials benar
- Check internet connection
- Check logs untuk detail error

ISSUE: Rate limit error (429)
SOLUSI:
- Tunggu 1 menit
- Limit: 100 requests per 60 detik
- Normal untuk production security
